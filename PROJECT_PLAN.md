# خطة نظام إدارة المخزون والمستودعات WMS

## 1. الفلسفة

النظام ليس ERP وليس Accounting وليس POS. النظام متخصص في إدارة المخزون والمستودعات بدون أي أسعار أو محاسبة.

القاعدة الأساسية:

```text
CurrentStock = الكمية الحالية
StockMovements = التاريخ الكامل لكل حركة
```

ولا تتغير الكمية إلا بعد اعتماد مستند تشغيلي.

## 4. قواعد العمل

- لا تعديل مباشر للكمية.
- لا كميات سالبة.
- لا تعديل مستند معتمد.
- كل حركة مخزون لها مرجع.
- Batch مطلوب فقط عندما يتطلبه الصنف.
- Serial مطلوب فقط عندما يتطلبه الصنف.
- Expiry مطلوب فقط عندما يتطلبه الصنف.
- الإلغاء يتطلب سببًا وسجل نشاط.
- التعطيل بدل الحذف.
